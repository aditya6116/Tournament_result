Project : Tournament Result

Project Libraries used : python, psql.

Description : The project is mainly to test tournament results using a 
	      database.
             
How to run : 1) Download the Latest version of virtual machine.

             2) Download and install Vagrant.

             3) Clone or Download the VM configuration files.

'            4) Run the command 'vagrant up' in your terminal.               
             5) Run the command 'vagrant ssh'.

	     6) Change the directory to '/vagrant/tournament'.

	     7) Enter the PSQL command interface using the command                 'psql'.
	     8) Import the database config file with command '\i                 tournament.sql'.

	     9) Run the tournament.py with python in terminal.

             10) Go back to the terminal and run the python files                          'tournament_test.py'.
